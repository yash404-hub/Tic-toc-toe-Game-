# 🕹️ Tic-Tac-Toe Game

A simple and fun implementation of the classic **Tic-Tac-Toe** game built using **HTML**, **CSS**, and **JavaScript**.

🎮 The goal is to get three in a row — horizontally, vertically, or diagonally — before your opponent!

---

## 🚀 Features

✔️ Two-player gameplay (X vs O)  
✔️ Win and draw detection  
✔️ Clean and simple UI  
✔️ Runs directly in the browser
